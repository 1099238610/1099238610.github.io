布隆过滤器 Bloom Filter

原理

布隆过滤器是用 Bitmap 和 Hash 函数组成的数据结构, 主要用于检索在大量数据中是否存在某个数据, 当一个数据存入布隆过滤器时, 会有几个步骤:

  1. 对元素进行多个 hash 计算, 会得到多个哈希值 n1, n2 …
  2. 查询 Bitmap 中索引为 n1, n2… 的数据是否有 0, 有的话则数据不存在
